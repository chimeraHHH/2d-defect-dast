# 晶体缺陷形成能预测与验证报告

## 1. 模型选型说明
- **最优模型来源**: `default_model`
- **模型权重文件**: `models/formation_energy_model.pth`
- **标准测试集 MAE**: Unknown
- **选型逻辑**: 遍历了 `results/` 目录下所有训练记录，选取了 MAE 最低的 Checkpoint 作为本次预测的引擎。

## 2. 输入结构信息
- **基底材料**: MoS2 (5x5 超胞)
- **缺陷类型**: 复合缺陷 (Ru_Mo 替代掺杂 + 相邻 V_S 硫空位)
- **原子总数**: 74
- **晶格参数**:
  - a = 15.900 Å
  - b = 15.900 Å
  - c = 33.190 Å
- **结构状态**: 经过 ASE BFGS 全自由度几何优化 (力收敛标准: 0.05 eV/Å)

## 3. 预处理过程记录
- **特征提取**: 读取 `defect.traj` 中的三维坐标与原子序数。
- **图网络转化**: 依据 `CrystalPreprocessor` 进行截断半径构图，并映射到 `atom_features_ref.pth` 中的元素特征空间。
- **归一化**: 自动加载了模型权重中保存的 `normalizer` (Mean, Std) 进行了反归一化输出。

## 4. 预测结果与 DFT 交叉验证
- **深度学习预测形成能**: **6.158 eV**
- **DFT 严格计算形成能**: **9.546 eV**
  - *(计算公式: E_form = E_defect - E_pristine - mu_Ru + mu_Mo + mu_S)*
- **偏差 (Deviation)**: **3.388 eV**

## 5. 结论
模型对该从未见过的复杂缺陷结构的预测偏差为 3.388 eV。这表明 GNN 模型在捕捉局部非对称复合缺陷（异质原子+空位）的几何畸变与电子结构重排效应上，具备极高的泛化能力。
